‚ÑπÔ∏è Setup Instructions

  1. Plug in VK-162 GPS: Connect USB GPS dongle to Raspberry Pi
  2. Install GPS software: Run setup script on Pi: bash ~/stream-control/setup_gps.sh
  3. Start GPS reader: The GPS data service will start automatically
  4. Wait for GPS lock: Takes 30-60 seconds outdoors with clear sky view
  5. Configure overlays: Enable desired data elements above
  6. Save settings: Click "Save Overlay Settings" below
  7. Start streaming: Overlays will automatically appear on your stream!
